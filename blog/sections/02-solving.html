Can You Play Optimal Yatzy?

The game has a finite state space that can be fully enumerated. The state of a Yatzy game is determined by two things: which categories have been scored (a 15-bit bitmask, giving 32,768 possibilities) and how much progress you've made toward the upper section bonus (0 to 63 points, capped). That's 32,768 × 64 = 2,097,152 unique game states.

Using backward induction — starting from the end of the game and working backward — we can compute the exact expected value of every state. At each state, the algorithm considers all possible dice outcomes, all possible keep/reroll decisions, and all possible category assignments. The result is an 8 MB lookup table that answers the question: "given where I am in the game and what I rolled, what should I do?"

An optimal player scores 248.4 points on average — about 66% of the theoretical maximum. The standard deviation is 38.5, meaning most games fall between 210 and 290 points. The entire precomputation takes 2.3 seconds on a modern laptop.

An instructive failure mode: if you naively maximize the immediate score each turn (always pick the category that scores highest right now), you average only 118.7 points — less than half of optimal. The greedy approach destroys the upper section bonus by filling categories sub-optimally, which cascades through the rest of the game.
